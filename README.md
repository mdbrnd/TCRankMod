# Taeko & Co Rank Mod

## Setup

This mod allows for different roles such as visitor, helper, member and admin.<br />
Commands:<br />
/myrank<br />
/myinviter<br />
Op only:<br />
/setrank \<person\> \<rank\><br />
/setinviterof \<person\> \<person\>
